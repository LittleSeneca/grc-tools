Open-source GRC (Governance, Risk, and Compliance) policy templates, tools, and guidance. Built from real-world deployment experience and aligned with NIST 800-53, SOC 2, and ISO 27001.

## Start here

If you are building a security program from scratch, read the [Getting Started guide](Guides/Getting-Started).

## What's here

### Policy Templates

[32 production-ready policy templates](Policy-Templates/) in ISP format. Every policy comes with:

- A fill-in-the-blanks template that defines what is required
- A companion procedure document that describes how to implement it
- A README with plain-English explanation, gotchas, and implementation advice

## How to use

1. Read the [Getting Started guide](Guides/Getting-Started) to determine your tier
2. Browse [Policy-Templates](Policy-Templates/) and find the policies you need
3. Read the README in each policy folder first
4. Copy the template and replace all `____` placeholders
5. Review with legal before publishing

## Frameworks mapped

NIST SP 800-53 Rev 5, SOC 2 (Trust Services Criteria), ISO 27001:2022 (Annex A), NIST CSF 2.0, NIST SP 800-63B
