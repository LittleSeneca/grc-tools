# Control Self Assessment Process

Effective Date: ____

## Purpose

This document defines the process for conducting Control Self Assessments (CSAs) to evaluate the design and operating effectiveness of security and compliance controls within ____.

## Scope

This process applies to all controls defined in the organization's Information Security Program (ISP) and any additional controls required by regulatory frameworks, contractual obligations, or internal policy.

## Background

Control Self Assessment is a management technique where control owners evaluate the effectiveness of their own controls. CSAs provide early warning of control weaknesses, promote ownership of control performance, and reduce reliance on external audits as the sole detection mechanism.

## Process

### 1. Control Inventory Maintenance

- Maintain a complete inventory of all controls in the ISP, mapped to relevant frameworks (NIST 800-53, SOC 2, ISO 27001).
- Each control must have a designated owner.
- The control inventory must be reviewed and updated at least ____ (quarterly/annually).

### 2. Assessment Scheduling

- CSAs are conducted on a ____ (quarterly/semi-annual/annual) basis.
- High-risk controls may be assessed more frequently.
- New controls must be assessed within ____ days of implementation.
- Controls with recent failures or incidents must be prioritized for reassessment.

### 3. Assessment Execution

Each control owner must evaluate their control against the following criteria:

- **Design Effectiveness:** Is the control designed to achieve its stated objective?
- **Operating Effectiveness:** Is the control operating as designed?
- **Evidence Availability:** Can the control owner produce evidence of the control's operation?

Assessment results must be documented using the standard CSA worksheet.

### 4. Assessment Ratings

| Rating | Criteria |
|--------|----------|
| **Effective** | Control is properly designed and operating as intended with available evidence |
| **Needs Improvement** | Control is generally effective but has minor gaps in design, operation, or evidence |
| **Ineffective** | Control has significant gaps requiring remediation |

### 5. Remediation Tracking

- Controls rated "Needs Improvement" must have a remediation plan documented within ____ days.
- Controls rated "Ineffective" must have a remediation plan documented within ____ days and escalated to the ____ (Oversight Body).
- Remediation plans must include specific actions, owners, and target completion dates.
- Remediation progress must be reviewed at each oversight meeting.

### 6. Reporting

- CSA results must be compiled into a summary report for the Oversight Body.
- The report must include: overall control health metrics, controls by rating, open remediation items, and trends over time.
- CSA reports must be retained for a minimum of ____ years.

## CSA Worksheet Template

| Field | Description |
|-------|-------------|
| Control ID | Unique identifier from control inventory |
| Control Title | Name of the control |
| Control Owner | Person responsible for the control |
| Framework Mapping | NIST 800-53, SOC 2, ISO 27001 references |
| Assessment Date | Date of assessment |
| Design Rating | Effective / Needs Improvement / Ineffective |
| Operating Rating | Effective / Needs Improvement / Ineffective |
| Evidence Reviewed | Description of evidence examined |
| Gaps Identified | Description of any gaps or weaknesses |
| Remediation Plan | Actions, owner, target date (if applicable) |
| Overall Rating | Effective / Needs Improvement / Ineffective |

## Roles and Responsibilities

| Role | Responsibility |
|------|----------------|
| Control Owners | Execute CSAs for assigned controls; maintain evidence; implement remediation |
| ____ (CISO/Security Officer) | CSA program management; reporting to Oversight Body; quality review |
| Oversight Body | Review CSA results; ensure remediation is tracked |

## Compliance and Enforcement

Failure to complete CSAs on schedule or to remediate identified control deficiencies may result in escalation to executive management and potential impact on audit opinions.
